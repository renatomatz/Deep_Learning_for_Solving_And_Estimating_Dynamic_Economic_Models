# AGENTS.md

## Purpose

Lecture 18 is the course wrap-up: synthesis, decision guidance, and outlook
across DEQNs, PINNs, surrogates, SMM, climate IAMs, and deep UQ.

## Map

- Start with `README.md`.
- `slides/lecture_18_wrap_up.tex` is the slide source.
- `slides/fig/` contains a large reused figure set from earlier climate,
  surrogate, and policy lectures.
- There is intentionally no `code/` or `code_julia/` directory.

## Editing

Treat this lecture as a synthesis deck. If changing claims about method choice or
course flow, check `../../COURSE_MAP.md` and the relevant earlier lecture README.

Do not add runnable examples here unless the user explicitly asks; route code
examples to the lecture where the method is taught.

The Julia translation track has no local Lecture 18 notebook by design. Do not
invent Julia/Lux/Pluto wrap-up notebooks or placeholder `code_julia/` material
when updating synthesis guidance; refer back to the lecture-specific previews in
Lectures 13-16 when Julia context is needed.

Several climate figures are duplicated from earlier lectures. The current wrap-up
deck uses only a small subset of `slides/fig/`, while the folder also contains
copied extras and the typo asset `damge.png`. Check `../../assets/attributions.yml`
and update duplicate copies consistently.
